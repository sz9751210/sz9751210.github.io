Go syntax

👨‍💻簡介 在這篇文章裡，會簡單介紹幾個關鍵的基本概念和語法結構，加快上手這門程式語言。 文件組成 一個典型的 Go 語言程式包含以下幾個部分： Package 宣告 程式的開頭會包含 package 宣告，用來指定當前的檔案屬於哪個程式套件（package）。 package main 匯入套件 我們可以透過 import 關鍵字引入需要使用的套件（packages）。這些套件提供了各種功能，讓我們能夠輕鬆地進行開發。 import ( "fmt" ) 函式定義 在 Go 語言中，程式邏輯主要是由函式（functions）組成的。以下是一個主要函式 main 的定義，它是每個 Go 程式的進入點。 func main() { fmt.Println("Hello World!") } 語法注意事項 在撰寫 Go 程式時，有幾點語法上的注意事項需要記住： 語句可以透過換行或分號（;）分隔。 左大括號 { 不能單獨出現在一行的開頭，應該與其他語句在同一行。 註解 在程式碼中，註解可以提供有關程式邏輯的解釋和說明。Go 語言支援兩種註解格式： 單行註解： // 這是單行註解的範例 多行註解（C 風格）： /* 這是多行註解的範例， 可以跨越多行並提供更多細節說明。 */ 📚Reference Go Syntax (w3schools.com)
